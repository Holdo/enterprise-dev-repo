Welcome to our promising Enterprise-dev-repo project

Ever needed to store your development artifacts, version them and perform searches on them? Then you came to the right place. Our project intends not only to deliver these functions needed by the Enterprise Middleware industry, but to deliver them in a modern, high-speed and reliable fashion.

Technology behind the curtain

Red Hat's Undertow web server winning performance benchmarks combined with WebSocket protocol promises fast web interaction. Embedded BaseX XML database on the backend provides standardized XQuery document access and good scalability. We didn't forget about user experience as well... that's why we've bet on a modern OpenUI5 MVC JavaScript framework built upon JQuery, which supports responsive one page client side applications.
